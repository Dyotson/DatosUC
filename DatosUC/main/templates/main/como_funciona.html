¿Como funciona Datos_eUC

Un pequeño resumen del funcionamiento de Datos_eUC

¿Como funciona en detalle todo?

1. El bot o data_collector crea un excel/csv de la información que fue recoletectada

2. La información recolectada se pasa por un filtro y es procesada para ser subida a la base de datos

3. La información de la base de datos es analizada y rectificada por un equipo de personas

4. La base de datos envia la información a la pagina web
